# Problem 1: Highlight Menu

**Description:**  
A simple terminal-based Employee Form menu where the user can navigate options using arrow keys. The selected option is highlighted.  

**Features:**  
- Arrow Up / Down to move the highlight.
- Enter to select an option.
- ESC to exit the program.
- Loops menu continuously until Exit is selected.

# Problem 2: Student Grades 2D Array

**Description:**  
Stores student information and their grades for multiple subjects using structs and a 2D array. Calculates sum per student and sum & average per subject.

**Features:**  
- Enter name, age, and grades of students.
- Display total grades per student.
- Display sum and average of grades per subject.
